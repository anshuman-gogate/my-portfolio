The pacman game

classic pacman game

Pacman game using HTML, CSS and JavaScript

Classic pacman game, we have pacman and 4 ghosts that will try to eat pacman. Pacman can also eat ghosts but for this he needs to eat power pallets and then has 10 seconds to eat them. You need to score 500 points to win the game. These points can be scored by eating pacdots(1 point), power palletes(10 points) and ghosts(100 points).

This project helped me to use the concepts of JavaScript like classes, etc to our benifit.

I plan to optimize this game to make it faster and employ a shortest path algorithm so that ghosts chase pacman.
