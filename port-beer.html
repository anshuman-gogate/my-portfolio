The Beer App

beer app using punk api

Beer app using HTML, CSS, JavaScript and Punk API

This beer app shows you different beers and also the information about each beer(you need to hover on the beer card). You can also filter beers by ABV and IBU plus also move across various pages. All the information is taken from Punk API.

I have used fetch() API, asycn/await, Flexbox, Grid, etc in this project.
